Front End

Front End means the visible part of a website or application that users see and interact with. It is also called Client-Side Development.

Main Technologies Used in Front End

  1. Main Technologies Used in Front End

    • Used to create the structure of a webpage.
    • Headings, paragraphs, images, forms

  2. CSS (Cascading Style Sheets)

    • Used to design and style the webpage
    • Colors, fonts, layout, spacing

  3. JavaScript

    • Used to add functionality and interactivity
    • Button click actions, form validation, sliders

Back End

Back End means the server-side part of a website or web application. It works behind the scenes and is not visible to users.If you create an HTML page that is Front End.When data is saved, login is checked, or database is connected — that is Back End.
