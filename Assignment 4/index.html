My Skills

HTML

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML defines the structure of a website using elements like headings, paragraphs, images, links, forms

CSS3

CSS3 (Cascading Style Sheets) is used to style and design web pages. It controls the layout, colors, fonts, spacing, and animations of HTML elements. CSS3 helps make websites more attractive and responsive for different devices like mobiles, tablets, and desktops

JavaScript

JavaScript is a programming language used to make web pages interactive and dynamic. It allows developers to add features like form validation, animations, pop-ups, and dynamic content updates. JavaScript works together with HTML and CSS to create modern and responsive websites.

Responsive Design

Responsive Design is a web design approach that makes a website look good and work properly on all devices such as mobiles, tablets, and desktops. It uses techniques like flexible layouts, images, and CSS media queries to adjust the website layout based on the screen size.
